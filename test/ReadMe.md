# CCIP Cross-Chain Name Service

This repository contains resources developed as part of the CCIP Bootcamp. These resources are designed to work with the CCIP Cross-Chain Name Service.

## Prerequisites

- **Node.js**: Ensure Node.js is installed. Use version 18, as this was required by Hardhat during the development process.
  - Install version 18 with `nvm use 18`.
- **Git**: Ensure Git is installed for cloning repositories.

## Setup Instructions

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/UdaySingh131105/ccip-cross-chain-name-service.git
   ```

2. **Change Directory**:

   ```bash
   cd ccip-cross-chain-name-service
   ```

3. **Install Dependencies**:

   ```bash
   npm install
   ```

4. **Run Tests**:

   Execute the test suite with the following command:

   ```bash
   npx hardhat test --network localhost
   ```

   OR

   ```bash
   npx hardhat test --network hardhat
   ```
